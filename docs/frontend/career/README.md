# 职业规划
此处所列发展方向及成长阶段摘抄自第三方，在成长过程中将进行适当调整。拥有一个良好的职业规划及学习路线对于长期工作及学习来说至关重要，将指引我们不断向前。这里主要讲技术骨干方向，根据每个人发展方向技术深度可能有所差异，但都是为了实现自身价值同时获得成就感，共勉。

**发展方向**
* 初级前端开发工程师
  * 静态页面布局
  * 基础功能开发
* 中级前端开发工程师
  * 普通的日常需求
  * 普通的前端项目开发
* 高级前端开发工程师
  * 完成前端项目开发
  * 独立完成需求解决方案
  * 组件或类库编写
* 资深前端开发工程师
  * 提供前端解决方案
  * 负责前端代码审核和规范制定
  * 保障项目进度和质量
* 前端技术专家
  * 提供技术解决方案
  * 设计前端架构及组件开发
  * 技术专利的发掘和专利申请的初步审核

**成长阶段**
* [第一阶段：JavaScript 深度剖析](/frontend/career/level1.md)
  * ECMAScript 新特性
  * JavaScript 异步编程
  * TypeScript 高级编程
  * 函数式编程范式
  * JavaScript 性能优化
* [第二阶段：前端工程化实战](/frontend/career/level2.md)
  * 脚手架工具
  * 自动化构建
  * 自动化测试
  * 自动化部署（CI / CD）
  * 模块化开发与 Webpack
  * 规范化标准
* [第三阶段：核心框架原理与进阶](/frontend/career/level3.md)
  * Vue.js 原理深度剖析
  * Vue.js 高级与进阶
  * React 设计原理解密
  * React 进阶与实战
  * Angular 企业实战开发
* [第四阶段：Node.js 全栈开发](/frontend/career/level4.md)
  * Node.js 高级编程
  * NoSQL 数据库
  * GraphQL API 开发
  * 企业级框架
* [第五阶段：泛客户端开发](/frontend/career/level5.md)
  * 小程序与快应用
  * Hybrid App 开发
  * React Native
  * Flutter 原生 App 开发
  * Electron 桌面应用开发
* [第六阶段：商业级技术解决方案](/frontend/career/level6.md)
  * Serverless 无服务端方案
  * 中途岛 / 中间层方案
  * 首屏性能提升方案
  * 数据埋点方案
  * 长列表无限滚动方案
  * API 接口鉴权方案
  * 更多常见方案
* [第七阶段：高阶技术专题](/frontend/career/level7.md)
  * 微前端架构与实践
  * PWA 渐进式 Web 应用
  * 数据可视化
  * 现代化 Web 101 架构剖析
  * Web Components
  * 更多技术专题
